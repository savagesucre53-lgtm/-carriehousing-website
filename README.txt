CARRIE HEN REALTY GROUP — EMAIL-GATED APPLICATION VERSION

WHAT CHANGED
- The public online rental application was removed from the website.
- Visitors must email carriehen@protonmail.com to request an application.
- The email button pre-fills the information the applicant should provide.
- The website explains that payment instructions are sent only after property availability is confirmed.
- The application form is sent manually after the application-fee payment is verified.
- Security deposit, first month's rent, lease signing, and key-release steps are clearly explained.
- A detailed seven-step rental process and FAQ section were added.

RECOMMENDED WORKFLOW
1. Applicant emails the request.
2. Confirm the exact property is available.
3. Send the written fee disclosure and official payment instructions.
4. Confirm payment.
5. Email the rental application PDF from assets/rental-application.pdf.
6. Review the completed application.
7. If approved, send the lease and itemized move-in funds statement.
8. Confirm signed lease and funds before releasing keys.
